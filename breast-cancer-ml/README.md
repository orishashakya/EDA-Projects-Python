# Project Overview
This project aims to predict whether a tumor is malignant or benign using the Framingham dataset from Kaggle. It demonstrates data analysis, visualization, and machine learning modeling using Python libraries. The project highlights how raw healthcare data can be transformed into actionable insights using logistic regression.
